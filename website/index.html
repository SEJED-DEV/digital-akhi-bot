Frequently Asked Questions

Find answers to common questions about setting up, pricing, security, and verified databases.

The bot queries authenticated Islamic databases with precise grading metadata. When a user runs a lookup, the bot displays precise gradings (Sahih, Hasan, Da'if) and specific collection context references.
BYOK allows server administrators or individual members to input their own AI provider keys (like OpenRouter or OpenAI) using the secure /setkey command. Your custom key is stored with robust AES-256-CBC encryption. Using your own key removes all rate limits!
Every new user can test the bot's AI features for 48 hours using our default API keys. After this period, you will be required to add your own API key (BYOK) via /setkey to continue using the AI. Non-AI commands like /prayer remain free forever.
If our primary AI model service encounters latency or temporary outages, the bot's dynamic client-side handler automatically shifts queries to secondary backup providers within milliseconds, ensuring absolute stability.
