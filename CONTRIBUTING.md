# Contributing to Islamic Community Discord Bot

First off, thank you for considering contributing to this project! It's people like you who make the bot better for the entire community.

## Code of Conduct

By participating in this project, you are expected to uphold our Code of Conduct:
- Be respectful and inclusive.
- Use professional language.
- Focus on what is best for the community.

## How Can I Contribute?

### Reporting Bugs
- Use the GitHub Issues tracker.
- Provide a clear and descriptive title.
- Include steps to reproduce the issue and your environment details.

### Suggesting Enhancements
- Open a GitHub Issue with the tag `enhancement`.
- Describe the feature and why it would be useful.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. Ensure your code follows the existing style and conventions.
3. Verify that your changes compile and pass any existing tests.
4. Update the documentation if necessary.
5. Submit your PR with a clear description of the changes.
